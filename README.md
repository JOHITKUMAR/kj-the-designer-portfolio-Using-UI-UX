## 🚀 Getting Started

### Run locally

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```
2. Open the main page file directly in your browser to preview it.

## 🌐 Deploying to GitHub Pages

1. Push your code to a GitHub repository.
2. Go to your repo → **Settings** → **Pages**.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or `master`) → `/ (root)`
4. Click **Save**. Your site will be live at:
   ```
   https://<your-username>.github.io/<repo-name>/
   ```
5. It can take a minute or two for the site to go live after the first deploy.

> 💡 If your main HTML file isn't named `index.html`, rename it — GitHub Pages looks for `index.html` at the root by default.
